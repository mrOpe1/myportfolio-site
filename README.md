## 🛠️ Tech Stack

### Frontend
- **HTML5 & CSS3:** Semantic structure and modern layouts (CSS Grid/Flexbox).
- **JavaScript (ES6+):** Custom interactive logic and DOM manipulation.
- **Typography:** Inter & Space Grotesk (Google Fonts).
- **Icons:** FontAwesome 6.

### Design Tools
- **Figma:** For UI/UX wireframing and high-fidelity prototyping.
- **Adobe XD:** Design and asset creation.

### Backend & Expertise
- **Node.js & PHP:** Server-side logic.
- **MySQL:** Database management and design.
- **SEO & Performance:** Optimized for Core Web Vitals and search rankings.

---

## ✨ Features

- **Dual Theme Support:** Fully functional Dark and Light modes with local storage persistence.
- **Interactive Hero Section:** Dynamic typewriter effect and floating particle system for a premium feel.
- **Responsive Design:** A mobile-first approach ensuring a flawless experience on smartphones, tablets, and desktops.
- **Scroll Animations:** Sophisticated reveal animations using the `IntersectionObserver` API.
- **Project Gallery:** A filterable showcase of featured work with high-quality visual states.
- **Advanced Contact Form:** Custom animated dropdowns and integrated validation for smooth user interaction.
- **SEO Optimized:** Implemented meta tags and structured heading hierarchy for better search visibility.
